## 技术栈

- **渲染API**: OpenGL 4.5+
- **窗口管理**: SDL3
- **数学库**: Eigen3
- **资源加载**: SDL_image
- **着色器语言**: GLSL 450
- **序列化**: JSON (nlohmann/json)
- **测试**: Google Test

## 性能目标

- **帧率**: 稳定 60 FPS（1080p）
- **绘制调用**: < 1000 次/帧
- **内存占用**: < 2GB
- **加载时间**: < 5秒（中等场景）

## 已知限制

1. 当前仅支持 Windows 平台
2. 需要 OpenGL 4.5+ 支持
3. 阴影质量有限
4. 缺少物理引擎集成
5. UI 系统较基础
